# C Learning — Bro Code Full Course

Apprentissage du langage C en autodidacte à partir de la vidéo Bro Code (YouTube).

## Pourquoi le C ?
Le C est la base de l'exploit development et du reverse engineering.
Comprendre les pointeurs et la gestion mémoire, c'est comprendre pourquoi
les buffer overflows et autres vulnérabilités mémoire existent.
